TASKFLOW — a single-page web app (works offline, saves in the browser).

TOP: an "Today" banner showing tasks due today with a checkbox per item and a
live count ("3 of 7 done today").

THREE TABS across the top (click to switch): [Today] [All Tasks] [Habits].

TODAY tab: tasks due today, grouped by time of day (Morning / Afternoon /
Evening). Each task row has a checkbox, a title, and an optional due-time.
An "Add task" button opens a small form: Title, Due date, Due time (optional),
Repeat (dropdown: none / daily / weekly).

ALL TASKS tab: every task, filterable by a search box and a status dropdown
(all / open / done). Each row has an edit (pencil) icon-button and a delete
(trash) icon-button. Bulk "Export as JSON" and "Import JSON" buttons at the
bottom.

HABITS tab: a list of recurring habits (e.g. "Drink water", "Stretch") each
shown as a row of the last 7 days as small colored dots (green = done, gray =
not done). Clicking a dot toggles it. An "Add habit" button opens a form:
Name, Frequency (daily / weekdays / custom).

Settings (gear icon, top right corner): toggle dark mode, toggle notifications,
"Clear all data" (with a confirmation dialog).
